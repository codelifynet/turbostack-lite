# Troubleshooting Guide

This section covers common errors you may encounter while developing with TurboStack. Errors are categorized for easy navigation.

## Quick Fixes

Most issues can be resolved with these commands:

```bash
# 1. Regenerate Prisma client
bun run db:generate

# 2. Sync database schema
bun run db:push

# 3. Restart development server
bun run dev
```

## Development Tips

### Quick Database Reset

If you encounter persistent database issues:

```bash
# Drop and recreate database
bun run db:push --force-reset

# Or with Docker
docker-compose down -v
docker-compose up -d postgres
bun run db:push
```

### Checking Database State

```bash
# Open Prisma Studio
bun run db:studio
```

### Clearing Build Cache

```bash
# Clear Turbo cache
bun run clean

# Or manually
rm -rf node_modules/.cache
rm -rf .turbo
```

<Tip>
  When in doubt, run these commands in order: 1. `bun run db:generate` 2. `bun
  run db:push` 3. Restart dev server
</Tip>
