## Installation

<Steps>
  <Step title="Clone the Repository">
    ```bash
    git clone https://github.com/codelifynet/turbostack-lite.git
    cd turbostack
    ```
  </Step>

<Step title="Install Dependencies">
  ```bash bun install ```
  <Tip>Bun installs dependencies significantly faster than npm or yarn!</Tip>
</Step>

  <Step title="Configure Environment">
    Copy the example environment file:

    ```bash
    cp .env.example .env
    ```

    Update the `.env` file with your configuration:

    <Tabs>
      <Tab title="Required" icon="asterisk">
        ```bash
        # Database (PostgreSQL)
        DATABASE_URL="postgresql://user:password@localhost:5432/turbostack"

        # Security Keys (change in production!)
        JWT_SECRET="your-super-secret-jwt-key-min-32-characters"
        SESSION_SECRET="another-super-secret-session-key"

        # App URLs
        FRONTEND_URL="http://localhost:4100"
        NEXT_PUBLIC_API_URL="http://localhost:4101"
        NODE_ENV="development"
        ```
      </Tab>
      <Tab title="Email (Resend)" icon="envelope">
        ```bash
        # Get your API key from resend.com
        RESEND_API_KEY="re_your_api_key_here"
        FROM_EMAIL="noreply@yourdomain.com"
        ```
      </Tab>
      <Tab title="Payments (Polar)" icon="credit-card">
        ```bash
        # Get your keys from polar.sh
        POLAR_ACCESS_TOKEN="polar_at_xxxxx"
        POLAR_WEBHOOK_SECRET="polar_ws_xxxxx"
        ```
      </Tab>
    </Tabs>

  </Step>

  <Step title="Setup Database">
    Push the Prisma schema to your database:

    ```bash
    bun run db:push
    ```

    <AccordionGroup>
      <Accordion title="Don't have PostgreSQL?" icon="database">
        Start PostgreSQL with Docker:

        ```bash
        docker run -d \
          --name turbostack-postgres \
          -e POSTGRES_PASSWORD=postgres \
          -e POSTGRES_DB=turbostack \
          -p 5432:5432 \
          postgres:16
        ```

        Then use this connection string:

        ```bash
        DATABASE_URL="postgresql://postgres:postgres@localhost:5432/turbostack"
        ```
      </Accordion>
    </AccordionGroup>

  </Step>

  <Step title="Start Development Server">
    ```bash
    bun run dev
    ```

    <Check>
      All services will start automatically with Turborepo!
    </Check>

  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Port already in use" icon="triangle-exclamation">
    If you see `EADDRINUSE` error:

    ```bash
    # Find process using the port
    lsof -i :4100

    # Kill the process
    kill -9 <PID>
    ```
  </Accordion>

  <Accordion title="Database connection failed" icon="database">
    1. Make sure PostgreSQL is running
    2. Check your `DATABASE_URL` in `.env`
    3. Ensure the database exists:

    ```bash
    createdb turbostack
    ```
  </Accordion>

  <Accordion title="Prisma client not generated" icon="gear">
    If you see Prisma errors, regenerate the client:

    ```bash
    bun run db:generate
    ```
  </Accordion>

  <Accordion title="Bun installation issues" icon="bug">
    Try reinstalling Bun:

    ```bash
    curl -fsSL https://bun.sh/install | bash
    source ~/.bashrc  # or ~/.zshrc
    ```
  </Accordion>
</AccordionGroup>
